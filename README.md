# CBOR::Decode version 0.01

CBOR [RFC 7049](http://tools.ietf.org/html/rfc7049) format decoder in 100 lines
of source code

This is not production code. Released just4fun.

## INSTALLATION

To install this module type the following:

   perl Makefile.PL
   make
   make test
   make install

## DEPENDENCIES

  Encode
